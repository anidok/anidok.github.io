VirtualPad

A simple text and RTF editor • Jan 2017

This is a simple text editor written in Java using Swing framework. It also supports formatted text (RTF) and some basic tools are provided to modify the contents. Users can save their modified contents to a new .rtf file or can also save as plain text format. Shortcut keystrokes are also provided in the software to ease up user's extra effort.

RacquetBall

A double player 2D tennis game • Feb 2017

This is a simple 2D tennis game which can be played between two players. Each player has been assigned a couple of keys to control the racquet. To provide players a challenging game, the speed of the ball will keep increasing after each complete cycle, i.e, each time the ball hits racquet of player 1.
I used Java graphic library Swing and introduced the concept of threading to build this game.
